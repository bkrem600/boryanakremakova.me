A joint first-year project of a inventory system written in Python programming language. The program uses GUI and is created for an engineering company that manufactures aircraft components. When the user accesses the inventory system, they can create a new batch with components, which is stored in a json file. This is a joint second-year project that implements a dynamic data structure to maintain details of current projects for an IT company. It is divided into three steps. The first step uses a linked list to allow the user to add, find, display and remove projects. The second step uses an array list to add, display and remove team member from projects. The final third step combines the previous steps into a binary tree structure. Coding for this project can be found here. Additional project, presenting a flight information system: here.
